# 饿了么固定费用统计 - 更新日志

## v1.1.0 (2025-10-08)

### 🎯 重大功能更新

#### 订单识别与绩效计算逻辑重构
- **核心变更**: 从百分比绩效改为按单收费模式
- **订单识别**: 门店总结算金额=33.95元时，自动计为1单
- **绩效标准**: 固定7元/单
- **计算公式**: 绩效总额 = 总订单数 × 7元

### ✨ 新增功能

#### 1. 订单自动识别
```javascript
// 每个门店的总金额为33.95元，计为1单
const orderCount = (Math.abs(amount - 33.95) < 0.01) ? 1 : 0;
```

**识别逻辑**:
- 计算每个门店所有记录的结算金额总和
- 如果总和为33.95元（误差±0.01），计为1单
- 示例：35元 + (-1.05元) = 33.95元 → 1单

#### 2. 表格结构优化
- ✅ 新增"订单数"列：显示每个门店的订单数（0或1）
- ✅ 新增"合同开始时间"列：作为表格第一列
- ✅ 汇总行显示总订单数

#### 3. 绩效计算增强
- ✅ 显示"总订单数"：统计所有门店的订单总数
- ✅ 显示"绩效总额"：红色醒目显示，字体加大
- ✅ 按单收取标签：7元/单（只读输入框）
